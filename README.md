## 개요
`HTML`, `CSS`를 활용한 웹페이지를 제작 프로젝트.  

단말기의 유형, 화면 해상도, 뷰포트 너비 등에 따라 사이트의 스타일(CSS)적용하는 **미디어 쿼리(Media Query)** 를 사용한 반응형 웹. 

**반응형 분기점**
1. `0`px ~ `767`px  
2. `768`px ~ `1023`px  
3. `1024`px ~   
   
## 주제 
매거진 사이트를 **반응형 웹** 으로 제작하기
